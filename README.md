MP3 Copy Tool
====

パソコンで管理している m3uファイル（MP3プレイリスト）のMP3ファイルを
ファイル名を変更し、プレイ順の数値をファイル名つけて別フォルダへコピーします。  
  
m3uファイルに対応していないカーナビ等のSDカード用に作成しました。  
  
SDカードへコピー後、UMSSort でファイル名ソートを推奨
 


## 開発環境
 Microsoft Visual Studio Community 2015

## 必要ランタイム
 .NET Framework 4.5  

## 動作環境
 Windows 10 Pro x64 でしか確認してません。

## 使い方

①プログラムを立ち上げる  
②保存先のフォルダを選択  
③保存するファイル名を設定  
④*.m3uをプログラムにドラッグ＆ドロップ
